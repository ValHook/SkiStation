Pour compiler:
Inclure la librairie statique libIrrlicht.a
Préciser dans header search directory le dossier irrlicht contenant tout les .h du moteur graphique.

Si au lancement de la simulation en 3D une erreur de DLL apparait il faut recompiler irrlicht.dll pour vore ordinateur
Son code souce est disponible ici http://irrlicht.sourceforge.net/downloads/ (prendre a version 1.8)
Alternativement uiliser le .a et la dll de la version 1.6 peut aussi macher.

Sinon le programme se lance parfaitement bien sur n'impore quel ordinateur de l'école.
